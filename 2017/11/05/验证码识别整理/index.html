发表于 2017-11-05   |   分类于 Python

验证码识别的大杀器是tesseract君


tesseract指南中文版-link

但是它处理的能力有限, 对于验证码图片咱们必须先走一篇预处理流程后再扔给它. 安装tesseract也就是一行brew的事情
