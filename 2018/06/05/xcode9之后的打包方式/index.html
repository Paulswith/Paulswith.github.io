XCode8.3之前的打包

https://github.com/webfrogs/xcode_shell/blob/master/ipa-build
阅读链接上的这个脚本, 可以理解构建三部曲为:

  1. Clean build cache
  2. 编译出.archive
  3. xcrun打包为.ipa文件

第三步是和XCode9差异最大的地方, 先记住这里:

使用指南

  • 根目录下执行 xcodebuild -list, 把下面user-define这些配置下:
  • 项目根目录生成package文件夹, 将脚本和ExportOptions.plist放进去.
